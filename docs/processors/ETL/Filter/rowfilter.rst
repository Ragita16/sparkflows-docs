
RowFilter

Type
---------- 

transform

Class
---------- 

fire.nodes.etl.NodeRowFilter

Fields
---------- 

+---------------+------------------------+-----------------------------------------------------------------------------------------------------+
| Name          | Title                  | Description                                                                                         |
+===============+========================+=====================================================================================================+
| conditionExpr | Conditional Expression | The filtering condition. Rows not satisfying given condition will be excluded from output DataFrame |

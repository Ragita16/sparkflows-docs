Cast Data Type
==========

Fire Insights provide processors for Cast Data Type

----------------------------------------


.. list-table:: Cast Data Type Processors
   :widths: 30 70
   :header-rows: 1

   * - Title
     - Description
   * - Cast To Different Types-1
     - It casts specified columns to different datatypes as specified
   * - Cast To Different Types-2

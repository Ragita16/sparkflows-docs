REST API's in Fire
==================

Overview
--------

Sparkflows provides REST API for interacting with it.

Below are examples using CURL. The first step is to log in with your username and password and acquire the session id.

Swagger is also enabled and is available at http://<machine-name>:8080/swagger-ui.html

First Step : Login
------------------

As the first step, log into Sparkflows and get the session id.

In the below examples, the Sparkflows Fire server is running on the local machine at : localhost:8080

You can replace it with your own machine name and port.
